Requesting

js-web is using the axios libery to make http requests

GET Request

web.request.get('/some-uri')
web.request.get('/some-uri',params)

POST Request

web.request.post('/some-uri')
web.request.post('/some-uri',params)
web.request.post('/some-uri',{param1: 'value',param2: 'value'})

Request an image

You can download an image with the request.image function. 

web.request.image('/image-url','/assets/new-image.jpg')
