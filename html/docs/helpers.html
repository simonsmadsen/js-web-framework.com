Helper functions: 

// console.log(web.helpers)
{
  hash: [AsyncFunction: hash],
  compareHash: [AsyncFunction: compareHash]
}

Using the hash functions 

const helpers = web.helpers
const password = 'mypw'
const hash = helpers.hash(password)
if(helpers.compareHash('mypw',hash)){
  return web.redirect('/admin')
}
return web.back()
