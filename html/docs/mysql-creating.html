Creating (MYSQL)

The following examples is using this migration 

db.table('users',{
  id: 'id',
  name: 'string',
  created: 'datetime'
})

All example using following const definions. 

const web = require('js-web')
const users = db.table('users')

Create(obj)

Use the create(obj) function on the return object from the table function (the users const in this example)

const simonsId = await users.create({
  name: 'Simon Madsen',
  created: db.now()
})

Create() inside a route

Remember to use the await and async, because the create function is returning a Promise. 

web.route('/create-user/:name', async (input) => {
  const userId = await users.create({
    name: input.name,
    created: db.now()
  })
  console.log('user id: '+userId+' has been created');
  return web.redirect('/')
})
