Mail

Mail functions: 

const web = require('js-web')
web.htmlMail(to,subject,htmlFile,templateContext)
web.plainMail(to,subject,text)

Setup

To get the mail working you have to fill the .env file mail section. The section is default setup to gmail, so if you use gmail as stmp provider, you can write you email and password and it will work. 

mail_host=smtp.gmail.com
mail_port=587
mail_ssl=false
mail_user=name@gmail.com
mail_password=mypassword
mail_from=

Plain mail 

const web = require('js-web')
const emailBody = 'Hello, this is the email body'
web.plainMail('person@gmail.com','email from js-web',emailBody)

Html mail

The html mail, is using a html file as email body and its using the same templating as the htmlRoute() function. You can read more about templating here 

const web = require('js-web')
web.htmlMail(
  'person@gmail.com',
  'email from js-web',
  'html/mail/welcome.html',
  {name: 'Simon'}
)
