Deleting (MYSQL)

The following examples is using this migration 

db.table('users',{
  id: 'id',
  name: 'string',
  created: 'datetime'
})

All example using following const definions. 

const web = require('js-web')
const users = db.table('users')

How to delete

Use the delete(where) function on the return object from the table function (the users const in this example)

users.delete({name: 'Simon'})
users.delete({id: 1})

Soft deleting

js-web has soft deleting build in, all you need is a bit field on all your table and set the .env file correct. In the example all tables need to have a field named deleted with the type of bool.
The delete function will only set the boolean to true and not delete the row, and all reading function will automatic filter the deleted rows away. 

mysql_soft_delete=true
mysql_soft_delete_field=deleted
